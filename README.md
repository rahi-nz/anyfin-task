# Anyfin-Task


[Vercel Anyfin-Task](https://anyfin-task.vercel.app/)


# Usage

```bash
touch .env.development
```
add `FIXER_KEY="your own Fixer API key"` to `.env.development`

```
yarn install
yarn dev
open http://localhost:3030
```

# Run test

```bash
yarn test
```
